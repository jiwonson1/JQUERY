jQuery 개요

* jQuery란?

기존에 자바스크립트 언어 만으로 구현하기에
복잡했던 구문 작성들을 간소화하기 위해 "존 레식"에 의해 개발된
JavaScript 기반 라이브러리 (유용한 기능들의 모음집)
즉, JavaScript 언어로 유용한 함수나 내용들이 이미 만들어져있는(정의되어있는) 모음집
복잡한 자바스크립트 구문을 보다 짧고 쉽게 개발 가능
(적은 양의 코드로 빠르고, 풍부한 기능 제공)

>> jQuery 장점
1) DOM요소와 관련된 스크립트를 보다 쉽게 구현할 수 있음
2) AJAX(비동기식 통신), 이벤트 처리 등등 폭넓게 지원함
3) jQuery와 관련된 확장형 플러그인, 오픈 API등을 지원
(차트, 슬라이드, 애니메이션 등등)

* jQuery 라이브러리 연결 방법

head태그내부에 script태그로써 jQuery 관련 .js 파일 가져오기
단, 오프라인 / 온라인 방식이 있음
- jQuery 공식 사이트 : 공식사이트

1. jQuery 라이브러리 다운로드 받은 후 경로 지정 (오프라인)
- .js 파일을 내가 직접 다운로드 받아 저장한 후 해당 저장된 파일 경로를 작성하는 방법

2. CDN(Content Delivery Network)를 통한 연결 (온라인)
- 라이브러리를 제공하는 사이트 주소를 이용하여 파일 경로 지정

* jQuery 시작하기

jQuery 명령은 문서상에 dom요소가 다 로드(다 만들어지고나서)된 후에 실행되어야함
> 자바스크립트에서의 해당 내용은 

        window.onload = function(){
            실행내용;
        }

        단, javascript에서의 window.onload 구문은 

        해당 문서의 dom요소들도 다 로드되고 뿐만아니라 연동되어있는
        외부 문서들도 다 가지고 온 뒤에 실행됨 

        그리고 window.onload는 해당 문서에서 단 한번만 작성 가능
> jQuery에서의 해당 내용은 3가지 구문으로 제공 
        1) jQuery(document).ready(function(){
                실행내용;
           });
        
        2) $(document).ready(function(){  // 보통 제일 많이 씀
                실행내용;
           });
        
        3) $(function(){   // 제일 축약형
                실행내용;
           });
        
        jQuery에서의 해당 내용은 외부문서를 다 가져오기전에 
        해당 문서의 dom만 다 로딩되면 실행됨 

        그리고 여러번 기술가능함, 제한 없음
